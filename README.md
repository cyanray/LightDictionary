# Light Dictionary

![](./doc/logo.png)

一个快速、轻量、漂亮的 UWP 词典程序。

## 特性

- [x] Windows 11 Style
- [x] 本地词典
- [ ] 网络词典(使用第三方 API)
- [ ] 网络翻译(使用第三方 API)
- [ ] 笔记本(生词本)
